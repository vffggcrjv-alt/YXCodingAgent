# Greeting Helper

## When to use
Use this skill whenever the user needs:
- Personalized greetings for someone
- Welcome messages
- Birthday or holiday greetings
- Professional or casual greetings
- Opening lines for messages or emails

## How to generate greetings

1. **Ask for context** if not provided:
   - Who is the greeting for? (name, relationship, role)
   - What's the occasion? (birthday, welcome, thank you, casual, formal)
   - What tone should it have? (professional, friendly, casual, warm)

2. **Generate appropriate greetings** based on the context:
   - Include the person's name
   - Match the tone to the occasion
   - Keep it brief and natural
   - Consider cultural context if relevant

3. **Provide options** when appropriate:
   - Give 2-3 variations so the user can choose
   - Include both formal and casual options if tone is unclear

## Examples

**For a colleague:**
- "Hi [Name], hope you're having a great day!"
- "Hello [Name], good to see you!"

**For a birthday:**
- "Happy Birthday, [Name]! Wishing you an amazing year ahead!"
- "Celebrating you today, [Name]! Have a fantastic birthday!"

**For a new team member:**
- "Welcome to the team, [Name]! Excited to work with you."
- "Hi [Name], glad to have you join us!"

**For a formal setting:**
- "Good morning, [Name]. Thank you for your time."
- "Dear [Name], I hope this message finds you well."
